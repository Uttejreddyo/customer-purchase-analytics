## Business Objective
Analyze retail transaction data to understand customer purchasing behavior, basket-size dynamics, and price-per-item stability, and convert insights into actionable business recommendations.

## Project Artifacts
- Excel analysis with pivot tables
- Business Requirements Document (BRD) summarizing insights
- Tableau dashboard for interactive visualization

## Tools Used
- Excel
- Tableau

## Key Analysis Highlights
- Basket size vs price-per-item analysis
- Stability of pricing across basket sizes
- Transaction contribution by high-item baskets
- Behavioral insights across store types and demographics

## Key Insights
- Larger baskets show lower but more stable price-per-item values
- High-item baskets contribute significantly to total transactions
- Pricing variability decreases as basket size increases
